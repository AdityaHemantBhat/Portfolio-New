## What's This About?

I'm Aditya, a frontend developer from Goa, India. After working with WordPress and Drupal for a while, I decided to build my own portfolio from scratch to showcase my work and skills. This site is basically my digital home on the web.

## The Stack

- **React 19** - Because I love component-based architecture
- **Tailwind CSS v4** - Makes styling so much faster
- **Vite** - Lightning fast development server
- **Lucide React** - Clean, consistent icons

## What You'll Find Here

The site has everything you'd expect from a portfolio:

- About me section with my background
- My education and work experience
- Skills breakdown (both technical and tools I use)
- Project showcase with live demos
- Contact info and ways to reach me

## Cool Features I Added

### Smooth Animations

I spent way too much time getting the animations just right. Everything fades in, slides, and floats in a way that feels natural but not overdone.

### Work Modal

Instead of a boring grid, I created this sliding modal that showcases my projects. You can scroll through them horizontally on desktop or vertically on mobile.

### Contact Modal

Same idea for the contact section - it slides in from the left with all my contact details and social links.

### Responsive Design

Works great on everything from phones to ultrawide monitors. I tested it on my phone, tablet, and multiple desktop sizes.

### Fun Touches

- Hover effects on the "Let's Cook!" buttons (they have little smoke particles!)
- Background text animations
- Floating decorative elements
- Custom loading states

## Running It Locally

```bash
# Clone it
git clone https://github.com/AdityaHemantBhat/portfolio.git

# Install stuff
npm install

# Fire it up
npm run dev
```

Then open `http://localhost:5173` and you're good to go.
